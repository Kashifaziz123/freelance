Discount On Sale Order

Discount on Sale Order module facilitates the user to apply discount on sale order both orderline wise and globally on the whole order. Discount can be successfully applied on tax included as well as tax excluded amount. This module also allows the user to manage sale order and invoice report with the discount applied.

Features:

  • Manage The Discount Type : Fixed/Percent.
  • Apply A Disocunt On The Orderline Of Fixed/Percent Type.
  • Apply A Global Discount On A Sale Order Of Fixed/Percent Type.
  • Apply The Global Discount On Order's Total Tax Included/Excluded Amount.
  • Manage The Sale Order Report With The Applied Discount.
  • Manage The Invoice Report With The Applied Discount.

  • Admin View

  • Change Log

Configuration

Enable The Global Discount Setting To Apply Global Discount On The Sale Order

Choose The Option For Applying Global Discount On Order's Total Tax Included/Excluded Amount

Apply Discount On Sale Order

Select The Discount Type : Fixed/Percent

Enter The Discount Amount

On The Basis Of Discount Type, Final Discount Amount Will Be Computed

Total Discount, Discount On Orderlines and Global Discount Will Appear Before Order Total

Invoice

Discount Type Is Managed In The Order Invoice

Total Discount Is Also Managed In Order Invoice

Sale Order Report

Sale Order Reports Are Managed With Discount Type

Sale Order Reports Are Managed With Total Discount Amount

Invoice Report

Order Invoice Reports Are Managed With Discount Type

Order Invoice Reports Are Managed With Total Discount Amount
